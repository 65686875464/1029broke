# my-app-tabs  Expo Tabs Template

This is a packaged Expo "tabs" template prepared for VibeCoding.

Contents:
- `app/`  Expo Router pages (tabs)
- `app.json`, `package.json`, `tsconfig.json`

Usage

1. npm install
2. npm start

Debug helper (optional)

This template supports an optional debug screen pattern. To enable the debug screen at runtime, set the environment variable `EXPO_DEBUG_SCREEN=true` when starting the app (only if the template includes the debug utility). Keep debug disabled in production.

Packaging notes

- This ZIP excludes `node_modules` to keep size small.
- The package includes `template.metadata.json` if present in the repo root.
